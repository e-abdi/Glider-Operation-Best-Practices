# Deployment

Deployment covers all operations from dock departure to handoff of the glider to the remote operations team. A methodical deployment reduces the risk of vehicle loss, ensures data continuity from the first dive, and protects crew safety during launch operations.

## Quick Reference

| Parameter | Value |
|-----------|-------|
| Typical Duration | 2–3 hours |
| Minimum Personnel | 3 (vessel crew + 2 glider operators) |
| Related Topics | [Mission Planning](../mission-planning/index.md), [Recovery](../recovery/index.md) |

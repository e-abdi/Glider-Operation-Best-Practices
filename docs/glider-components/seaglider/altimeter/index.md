# Altimeter

An underwater glider's altimeter is the primary instrument for preventing seabed strikes. It emits an acoustic pulse downward and measures the time-of-flight to the bottom, providing the glider's flight computer with a real-time altitude above the seafloor. When altitude drops below a programmed threshold, the glider commands an ascent.

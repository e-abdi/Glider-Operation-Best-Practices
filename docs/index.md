## How to Use This Site

- **Guides** provide step-by-step procedures with full context and troubleshooting.
- **Best Practices** capture institutional knowledge and the reasoning behind standards.
- **Checklists** are designed to be printed or saved as PDF and used in the field — click the **Print** button at the top of any checklist page.

Use the **search bar** (top right, or press `/`) to find any procedure by keyword.

!!! note "Contributing"
    This knowledge base is maintained as a Git repository. Click **Edit this page** at the top right of any page to suggest changes via GitHub.

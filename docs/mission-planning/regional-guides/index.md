# Regional Guides

Glider operations are shaped as much by regional conditions — currents, ice,
shipping traffic, fisheries, permitting regimes — as by the vehicle itself.
This section links out to published guides written by groups operating
gliders in a specific region, rather than trying to re-paraphrase
region-specific know-how into the rest of this site.

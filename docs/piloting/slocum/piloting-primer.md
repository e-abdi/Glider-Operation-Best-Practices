# G3 Piloting Primer

[:material-file-pdf-box: Download the primer (PDF)](papers/g3-piloting-primer.pdf)

!!! info "Source"
    *SL16 — User Notes — G3 Piloting Primer*, NorGliders / University of
    Bergen. A screen-by-screen companion to the
    [Mission Files](mission-files.md) and
    [Slocum Piloting Checklist](checklists/slocum-piloting-checklist.md)
    pages — it walks through the same territory from the SFMC UI rather
    than the underlying `.ma`/`.mi` files.

This is a visual, click-by-click walkthrough of piloting a Slocum G3 through
**SFMC** rather than a reference of concepts or parameters. It's the
document to open the first time you sit down at a new SFMC dashboard, not
the one to search for a specific parameter.

## What It Covers

- **Dashboard summary** — reading the map, comms plot, and the day-to-day
  checklist of what to glance at (last call-in, progress, XML script state,
  current waypoint).
- **Deployment details** — the summary/map, surfacings, device status, and
  surface sensor values views, with the key sensors worth plotting and what
  a healthy range looks like.
- **The glider terminal** — the control keys (`Ctrl-C`/`Ctrl-E`/`Ctrl-F`/
  `Ctrl-R`/`Ctrl-W`/`Ctrl-T`), sending files via `to-glider`/`to-science`,
  and pausing/resuming an XML script to send manual commands safely.
- **Waypoint creation** — the specific SFMC menu path for editing a mission
  plan and generating a new `goto` file, including a UI bug to route around
  (don't use *Options → Access Mission Plan*).
